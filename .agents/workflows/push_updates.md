---
description: Sync changes to GitHub after every modification
---
After completing any task, code edit, or improvement:
1. Run `git status` to verify changed files.
2. Run `git add .` to stage all changes.
3. Run `git commit -m "[Brief description of changes]"` with a clear and concise message.
4. Run `git push origin [current-branch]` to sync with the remote repository.
5. Verify the push was successful.
